19. Customization with Sass

  1. Initialize project: npm init -y
  2. Install Bootstrap source and Sass: npm i bootstrap sass
  3. Create src/_variables.scss and set overrides (see below).
  4. Create src/styles.scss that imports variables, then Bootstrap.
  5. Build CSS: npx sass src/styles.scss public/styles.css (or add an npm script).
  6. Link public/styles.css in your HTML instead of the CDN.
Example _variables.scss: 
// Exercise 19.2: Customize primary and border radius
$primary: #6f42c1; // purple
$border-radius: .75rem;
Example styles.scss: 
@use "variables"; // if you name it _variables.scss in same folder
@import "../node_modules/bootstrap/scss/bootstrap";
